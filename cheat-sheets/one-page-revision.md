# One-Page Revision Sheet: Core Concepts Summary

| Concept | Summary Definition | Core Example |
| :--- | :--- | :--- |
| **Globalization** | Integration of national economies into an interdependent global system. | Apple assembling global parts. |
| **Mercantilism** | Accumulating gold via trade surplus (Exports > Imports). | 17th-century European trade. |
| **Absolute Advantage** | Being more efficient at producing a good than any other nation (Adam Smith). | Brazil in coffee production. |
| **Comparative Advantage** | Producing at a lower opportunity cost relative to others (David Ricardo). | UK in software, India in IT support. |
| **WTO** | Multilateral organization regulating global trade rules. | Doha Development Round. |
| **IMF** | Financial institution maintaining exchange rate stability and providing emergency bailouts. | Greece debt crisis bailout. |
| **World Bank** | Institution focused on long-term economic development and poverty reduction via infrastructure loans. | Funding solar grids in Africa. |
| **Hedging** | Using financial contracts (forward/futures) to reduce foreign exchange transaction risk. | Buying forward contracts for euros. |
| **Incoterms** | Eleven standardized shipping terms defining delivery point, risk transfer, and cost division. | DDP, FOB, CIF. |
| **Bill of Lading** | Carrier receipt, contract of carriage, and title document for cargo. | Maritime shipping receipt. |
