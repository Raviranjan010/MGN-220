# LPU Exam Writing Strategy for MGN220

To secure an **A+ Grade (90%+ marks)** in LPU exams, follow this presentation system:

```
[ Question ] ──► [ Definition Box ] ──► [ Flowchart Diagram ] ──► [ Bullet Explanations ] ──► [ Real Case Study Example ]
```

## 1. Structure of the Answer (The 5-Step Formula)
1. **Definition Block**: Start with a formal academic definition. Enclose it in a box or blockquote using bold text.
2. **Visual Mapping**: Draw a quick diagram or flowchart showing the mechanism (e.g. Heckscher-Ohlin resource flow).
3. **Structured Explanations**: Write clear subheadings. Never write paragraph answers; use numbered lists and bullet points.
4. **Corporate Integration**: Mention at least one real-world corporate case (e.g. McDonald's glocalization, Apple's Foxconn assembly).
5. **Strategic Conclusion**: End with a brief summary statement.

## 2. Marks-to-Time Allocation (Based on 3-Hour Exam)
- **Section A (10 x 2 Marks)**: Spend 3 minutes per question (30 minutes total). Limit to 3 lines.
- **Section B (5 x 5 Marks)**: Spend 12 minutes per question (60 minutes total). Target 1 to 1.5 pages.
- **Section C (3 x 10 Marks)**: Spend 25 minutes per question (75 minutes total). Target 3 pages. Include large diagrams.
- **Section D (Case Study - 10/12 Marks)**: Spend 15 minutes. Link your answers directly to theories in the syllabus.
